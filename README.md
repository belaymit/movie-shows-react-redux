# 📖 movie-shows-react-redux
 <a name="about-project"></a>

**movie-shows-react-redux** is a project I created using ReactJS, redux and @redux-toolkit. From this project I learned:

> React 18 and best practices
> props and states
> React Router
> React Icons
> Re-Usable Components
> React Hooks (useState), use
> CSS transition and animation
> and more...

### Prerequisites

In order to run this project you need:

> Basic understanding of the following:
> HTM5/CSS3, SASS
> ES6
> webpack
> JSX
> ReactJS
> redux, react-redux, @redux/toolkit

### tools

> install Vs Code or any of your favored code editor
> install node
> react-router-dom
> redux, react-redux, @redux/toolkit 

```sh
 npx create-react-app .
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:belaymit/movie-shows-react-redux.git
```

### Install

Install this project with:

```sh
  cd movie-shows-react-redux
  npm install
```
